## 1. Why IPv6?

IPv4: ~4.3 billion addresses (running out!)
IPv6: 340 undecillion addresses (basically infinite)

```
IPv4: 192.168.1.1
IPv6: 2001:0db8:85a3:0000:0000:8a2e:0370:7334
```

## 3. IPv6 Types

```
Global Unicast:  2000::/3  (public, routable)
Link-Local:      fe80::/10 (local network only)
Loopback:        ::1       (like 127.0.0.1)
```

---

## 4. IPv4 vs IPv6

| Feature | IPv4 | IPv6 |
|---------|------|------|
| Length | 32 bits | 128 bits |
| Format | Decimal | Hexadecimal |
| NAT | Required | Not needed |
| Security | Optional | Built-in |
| DHCP | Required | Optional |
